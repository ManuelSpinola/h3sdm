# h3sdm 0.1.1

## New functions

* `h3sdm_pa_from_records()`: generates a presence/pseudo-absence dataset from
  user-provided records. Accepts a `data.frame` or `sf` object with coordinates
  in WGS84 (EPSG:4326). Supports optional filtering by `geospatialKosher` column,
  making it directly compatible with records downloaded via `bdcr_occurrences()`
  from the BiodataCR API.
